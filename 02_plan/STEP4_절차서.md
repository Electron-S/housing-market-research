# STEP 4: 수요 분석

## 목적
해당 권역의 실수요·투자수요 규모와 구조를 파악한다.
수요층이 누구인지 특정하고, 이들이 입지와 어떻게 연결되는지를 이 단계에서 명확히 한다.

## 입력
- STEP1_output.md (대상 범위, 분석 목적)
- STEP2_output.md (세제·대출규제 — 수요 억제·촉진 요인)
- `01_data/[target_id]/04_demand/` 원본 자료
- KOSIS 국가통계포털 (인구·가구·고용)
- 통계청 인구총조사
- 서울열린데이터광장 또는 지자체 통계

## 작업 체크리스트
- [ ] 인구·가구 수 및 최근 5년 증감 추이
- [ ] 고용·소득 수준 (배후 업무지구 연동)
- [ ] 유입·유출 인구 (전입·전출 통계)
- [ ] 자가/임차 비율
- [ ] 실수요층 특정 (연령·직업·소득 구간)
- [ ] 투자 수요 비중 추정 (다주택자 비율, 법인 매수 등)
- [ ] 수집 원본을 `01_data/[target_id]/04_demand/`에 저장
- [ ] 각 수치에 출처와 기준 시점 표기

## 산출물
`04_workspace/[target_id]_KR/STEP4_output.md`

### 포맷
```markdown
# STEP4 수요 분석

## 1. 인구·가구 현황
## 2. 고용·소득 환경
## 3. 유입·유출 동향
## 4. 실수요층 프로파일
## 5. 투자 수요 비중 추정
## 6. 수요 종합 판단
```
