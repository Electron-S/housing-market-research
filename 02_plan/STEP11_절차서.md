# STEP11: 보고서 집필

## 목적

STEP10_output.md의 장구성 설계를 기반으로 실제 보고서 초안을 완성한다.
독자(GS건설 수주·분양 실무팀)가 판단 근거를 바로 확인할 수 있는 두괄식 구조로 집필한다.

## 입력

- `STEP10_output.md` (장구성·핵심 메시지·집필 방향 확정본)
- `STEP1~STEP9_output.md` (분석 결과 전체)
- `02_plan/템플릿_아파트시장조사.md` 또는 `02_plan/템플릿_상가시장조사.md`

## 집필 순서

STEP10_output.md에서 확정된 장구성을 따르되, 아래 순서로 작성한다.

1. **사업 개요** — STEP0·STEP1 확정 정보 먼저 서술 (사실만)
2. **거시·정책 환경·가격·공급·수요·입지·경쟁·리스크** — STEP2~8 내용 재편
3. **결론 및 시사점** — STEP9 내용을 실행 문장으로 압축
4. **Executive Summary** — 결론과 핵심 수치만 뽑아 맨 앞에 배치 (마지막에 작성)

## 집필 체크리스트

- [ ] Executive Summary에 분양가 판단·수주 타당성 판단 선제 배치 (두괄식)
- [ ] STEP0 수주 전/후 분기 반영
  - **수주 전:** 수주 근거·리스크·제안 조건 포함
  - **수주 후:** 분양 실행안 포함
- [ ] 모든 수치에 출처·기준 시점 명시
- [ ] 사실·해석·판단 문장 분리
- [ ] 추정값은 `(추정)` 명시
- [ ] 광고성 문구 없음 확인
- [ ] 비교 단지 선정 근거 본문 포함
- [ ] 분양가 제안이 `약간 공격적` 기조인지 재확인
- [ ] 분량: 한국어 8,000~12,000자 (공백 포함) 목표
  - `count_chars.py`로 검증: `.venv/bin/python 03_code/count_chars.py [파일경로]`
