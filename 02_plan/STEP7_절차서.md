# STEP 7: 경쟁 비교

## 목적
대상지를 경쟁 단지와 항목별로 비교하여 강점과 열위를 명확히 정리한다.
비교를 통해 대상의 포지션을 확인하고, 결론의 차별화 근거를 마련한다.

## 입력
- STEP3_output.md (가격 수준)
- STEP5_output.md (경쟁 단지 목록)
- STEP6_output.md (입지 분석)
- 비교 단지 실거래가·분양가 정보 (국토부 실거래가, 청약홈)

## 작업 체크리스트
- [ ] 비교 대상 단지 선정 (통상 2~3개)
- [ ] 선정 근거 명시: 반경(통상 2km 이내)·규모·준공연도·생활권·수요층 유사성
- [ ] 항목별 비교표 작성: 가격·상품·입지·학군
- [ ] 비교 단지 최근 3개월 실거래가 평균 확인 (3.3㎡ 기준)
- [ ] 개발·분양 목적인 경우: 차등지수 산출 (아래 참조)
- [ ] 대상의 강점 정리 (경쟁 대비 우위)
- [ ] 대상의 열위 정리 (경쟁 대비 약점)
- [ ] 경쟁 단지 분양가·시세 출처와 기준 시점 표기

### 차등지수 산출 (개발·분양 목적 시)
비교단지 실거래가를 기준으로 대상지의 상대 가치를 보정하는 지수.
업계에서는 통상 입지·브랜드·연식·상품을 항목별로 평가해 보정율(±%)을 적용한다.

| 보정 항목 | 내용 | 방향 |
|---|---|---|
| 입지 | 역세권, 업무지구 접근성, 조망 | 우위면 + |
| 브랜드 | 시공사 브랜드 인지도 차이 | 약세면 − |
| 연식 | 신축 vs. 기축 | 신축이면 + |
| 상품 | Bay 구조, 평형 구성, 설계 품질 | 차별화되면 + |

> **평형 간 보정:** 대형평형(84㎡)은 중소형(59㎡) 대비 평당가가 통상 85~87% 수준.
> 59㎡ 기준으로 적정가를 먼저 산정한 후 대형평형은 별도 보정 지수를 적용한다.
