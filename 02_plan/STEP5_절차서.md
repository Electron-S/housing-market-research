# STEP 5: 공급 분석

## 목적
경쟁 물량과 미분양 현황을 파악하여 수급 균형 여부를 진단한다.
공급 과잉이면 가격 하방 압력, 공급 부족이면 분양성 우위의 근거가 된다.

## 입력
- STEP1_output.md (대상 범위, 기준 시점)
- STEP4_output.md (수요 규모)
- 국토교통부 입주예정 물량 통계
- 한국부동산원 R-ONE (미분양 현황)
- 지자체 정비사업 추진 현황
- 청약홈 (인근 사업지 청약 결과)

## 작업 체크리스트
- [ ] 해당 권역 입주예정 물량 (연도별, 최소 3년치)
- [ ] 현재 미분양 현황 (준공 후 미분양 포함)
- [ ] 정비사업(재개발·재건축) 추진 물량 및 시기 (지도 기반 정리 권장)
- [ ] 주요 경쟁 단지 목록 (분양 예정 포함)
- [ ] 인근 최근 분양 사업지 청약 경쟁률·완판 여부 (실수요 대기 수요 검증)
- [ ] 수요 대비 공급 비율 판단
- [ ] 각 수치에 출처와 기준 시점 표기

> **청약경쟁률 활용:** 동일 생활권 내 최근 분양 결과는 추상적인 수요 추정보다 신뢰도가 높다.
> 예: "인근 7구역 1순위 평균 242:1, 단기 완판" → 대기 수요 존재 확인
