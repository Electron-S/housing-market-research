# STEP 0: 사업 현황 사전 조사

## 목적
STEP1~9의 접근 방식과 정보 신뢰도 수준을 결정하는 사전 조사다.
사업 단계와 GS건설 포지션이 확인되지 않으면 확정 정보와 추정 정보가 혼재해 보고서 신뢰도가 낮아진다.
이 단계의 결과(수주 전/후, 확정 정보 범위)는 이후 모든 STEP의 전제값이 된다.

## 입력
- 사용자가 제시한 사업구역 명칭 또는 주소
- 관련 인터넷 검색 결과

## 작업 체크리스트

### 1. 사업 시행 단계 특정
- [ ] 사업 유형 확인: 재개발·재건축 / 지역주택조합 / 일반 도급 중 하나
- [ ] 해당 유형의 현재 진행 단계 특정

| 사업 유형 | 단계 목록 |
|---|---|
| 재개발·재건축 | 구역 지정 전 → 구역 지정 → 조합 설립 → 사업시행인가 → 관리처분인가 → 착공 → 준공 |
| 지역주택조합 | 조합원 모집 중 → 사업계획승인 신청 → 승인 완료 → 착공 → 준공 |
| 일반 도급사업 | 설계 중 → 인허가 중 → 착공 전 → 착공 후 |

### 2. 사업 주체 파악
- [ ] 조합 또는 시행사 명칭 확인
- [ ] 조합 설립 여부 확인
- [ ] 소유권 확보율 또는 동의율 확인 (정비사업의 경우)
- [ ] 시공사 선정 여부 및 현황 확인

### 3. GS건설 포지션 확인
- [ ] GS건설 수주 완료 여부 인터넷 검색으로 확인
- [ ] **수주 전이면:**
  - 입찰 준비 단계 여부 확인
  - 경쟁 건설사 입찰 현황 조사 (확인 가능한 범위)
- [ ] **수주 후이면:**
  - 설계 확정 여부 확인
  - 평형 구성·세대수 확정 여부 확인
  - 상가 포함 시: 연면적·층별 호실 구성 확정 여부 확인

### 4. 확정 정보 목록 작성
- [ ] 현 시점에서 확정된 정보 항목 목록화
- [ ] 추정 정보 항목 목록화 (이후 STEP에서 `추정`으로 명시할 항목)

### 5. 보고서 목적 분기 결정
- [ ] **수주 전:** 보고서 목적 = 수주 타당성 판단 + 제안 전략 수립으로 고정
- [ ] **수주 후:** 보고서 목적 = 분양·임대 전략 수립 + 실행 가능 분양가 제안으로 고정
