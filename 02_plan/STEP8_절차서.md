# STEP 8: 리스크 정리

## 목적
하방 리스크를 원인-영향 구조로 정리하여 결론의 조건부 판단 근거를 마련한다.
리스크를 나열하는 것이 아니라, 각 리스크가 대상 시장에 미치는 구체적 영향까지 서술한다.

## 입력
- STEP2_output.md (정책·금리 환경)
- STEP3_output.md (가격 민감도)
- STEP4_output.md (수요 구조)
- STEP5_output.md (공급 물량)
- STEP6_output.md (입지 특성)

## 작업 체크리스트
- [ ] 금리 상승 리스크: 원인·영향·발현 시기
- [ ] 공급 급증 리스크: 입주 집중 시기와 흡수 가능성
- [ ] 정책 변화 리스크: 대출규제·세제·규제지역 변경 가능성
- [ ] 경기 둔화 리스크: 고용·소득 감소가 수요에 미치는 영향
- [ ] 가격 조정 리스크: 현재 가격 수준의 고평가 여부
- [ ] 각 리스크를 낮음·중간·높음으로 등급화
