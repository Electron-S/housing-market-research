# STEP 9: 결론 및 시사점

## 목적
STEP1~8의 분석 전체를 행동 중심 판단으로 집약한다.
낙관·비관이 아니라, 조건을 명시한 판단문으로 작성한다.

## 입력
- STEP1~8 output 전체

## 작업 체크리스트
- [ ] 시장 종합 판단 (현재 수급·가격 위치 한 문장 요약)
- [ ] 행동 방향 제시: 매수·보유·매도·개발·보류 중 하나 + 근거
- [ ] 조건부 표현 사용 (예: "~한 경우 한해", "~를 전제로")
- [ ] 핵심 리스크 조건 명시 (STEP8 연동)
- [ ] 가격 또는 타이밍에 대한 구체적 시사점 포함
- [ ] 결론이 광고성 문구가 되지 않는지 점검
- [ ] 상가 조사인 경우: 정성 문장만으로 끝내지 않고 `층별 적정 평단가 밴드`를 표로 제시
- [ ] 상가 조사인 경우: `기준 임대료`, `목표 수익률`, `환산 평단가`, `보정 사유`를 함께 적시
- [ ] 상가 호실·면적이 미확정이어도 `1층 핵심 / 1층 비핵심 / 비1층 목적형 / 비1층 일반형` 기준의 예비 가격 밴드 작성

### 개발·분양 목적인 경우: 적정분양가 산출
STEP7의 비교단지 실거래가와 차등지수를 활용해 평형별 적정분양가를 도출한다.
단, 분양가 포지션은 보수적 하단값보다 시장 수용 가능한 상단을 소폭 선점하는 방향으로 잡는다.

**산출 순서:**
1. 비교단지 최근 3개월 실거래가 평균 (3.3㎡ 기준) 확인
2. 입지·브랜드·연식·상품 차등지수 적용 → 주력평형(통상 59㎡) 기준 적정 평당가 산출
3. 대형평형은 주력평형 대비 보정 지수(통상 85~87%) 별도 적용
4. 분양 조건 명시 (계약금 비율, 중도금 이자 후불/선납, 발코니 확장 포함 여부)
5. 상단 제안가와 방어 하한가를 함께 검토하되, 최종 권고는 상단에 더 가까운 `약간 공격적` 구간으로 제시

**표기 예시:**
```
전용 59㎡ 기준 15.2억 (@6,050만원/3.3㎡)
※ 조건: 발코니 확장 포함, 중도금 이자 후불제
```

### 상가 시장조사인 경우: 층별 적정 평단가 산정
상가는 아파트처럼 단일 평형 기준이 아니라 `층별 위치별 가격 밴드`로 제시한다.
호실 수나 전용면적이 확정되지 않았더라도 예비 가격 밴드는 반드시 산출한다.

**산출 순서:**
1. STEP3에서 확보한 `층별 임대료`와 `공실률`, STEP7에서 확보한 `비교 사례별 평단가·임대료·수익률` 확인
2. 목표 수익률 가정 설정
3. `예상 연 임대료 ÷ 목표 수익률`로 환산 매매가를 잡고 3.3㎡ 기준 평단가로 환산
4. 동선, 가시성, 전면폭, 코너 여부, 앵커 연계성, 역 직접 연결 여부를 반영해 층별 보정
5. `1층 핵심 / 1층 비핵심 / 비1층 목적형 / 비1층 일반형` 기준으로 하한·기준·상한 밴드 제시
6. 실제 호실 면적과 설계가 확정되면 예비 밴드를 호실별 가격으로 재산정

**필수 원칙:**
- 숫자 없는 `공격적/보수적` 표현만 적고 끝내면 안 된다.
- 비교 사례가 불완전해도 현재 확보 자료 기준의 `예비 밴드`는 반드시 남긴다.
- 숫자를 보류해야 한다면 `보류`만 쓰지 말고, 어떤 입력값이 없어 어느 범위까지 추정했는지 적는다.

**표기 예시:**
```markdown
| 구분 | 기준 임대료(월) | 목표 수익률 | 환산 평단가(3.3㎡) | 권고 밴드 | 비고 |
|---|---|---|---|---|---|
| 1층 핵심 호실 | XXX만원 | X.X% | @X,XXX만원 | @X,XXX~X,XXX만원 | 대로변, 주동선 |
| 1층 비핵심 호실 | XXX만원 | X.X% | @X,XXX만원 | @X,XXX~X,XXX만원 | 가시성 보정 |
| 2층 이상 목적형 | XXX만원 | X.X% | @X,XXX만원 | @X,XXX~X,XXX만원 | 메디컬·교육 전제 |
| 2층 이상 일반형 | XXX만원 | X.X% | @X,XXX만원 | @X,XXX~X,XXX만원 | 보수적 적용 |
```

## 결론 문장 형식 (예시)
```
분양성은 양호하며 초기 분양가를 시장 상단 대비 소폭 공격적으로 제시할 여지가 있다
실수요 기반은 강하나 경쟁 입주 물량이 집중되어 단기 속도 조절 필요
입지 우위는 분명하지만 상품 차별화 없이는 청약 흡수력 약화 가능
현재 가격 메리트 낮음 — 금리 안정 확인 후 재검토 권고
```
