# 프로젝트 개요 — 한국형 부동산 분석 워크플로우

> 기존 KCS-AI의 `분석 절차 + 중간 산출물 관리 + DOCX 리포트 생성` 구조를 재사용해, 한국 부동산 자산·지역·단지 분석에 맞게 재구성한 운영 개요다.

## 목적

부동산 자산 또는 권역을 대상으로 정량·정성 분석을 수행하고, 최종적으로 의사결정용 한국어 보고서를 생성한다.

대표 활용 예:

1. 아파트 단지 투자 검토
2. 생활권 시세 흐름 분석
3. 재건축·재개발 후보지 스크리닝
4. 상가·오피스·물류센터 수익성 비교
5. 분양사업 타당성 검토 초안

## 분석 대상

대상은 상장사가 아니라도 된다. 식별자는 자유롭게 정할 수 있다.

- 지역: `seongsu-office_KR`
- 단지: `songpa-helio-city_KR`
- 개발사업: `gimpo-newtown_KR`
- 자산군: `seoul-logistics-2026_KR`

## 핵심 질문

보고서는 보통 아래 질문에 답해야 한다.

1. 이 자산/지역의 가격 또는 임대료는 어떤 흐름인가
2. 수요와 공급은 어떤 방향으로 변하고 있는가
3. 입지 경쟁력은 무엇이고 약점은 무엇인가
4. 정책·금리·교통·개발호재가 어떤 영향을 주는가
5. 지금의 의사결정은 매수/보유/매도/개발/보류 중 어디에 가까운가

## 권장 평가축

| 평가축 | 확인할 내용 |
|---|---|
| 가격흐름 | 실거래가·호가·전세가·임대료 추세 |
| 수급 | 공급 물량, 공실, 미분양, 전입·전출 |
| 입지 | 교통, 학군, 업무지 접근성, 생활편의 |
| 정책 | 대출·세금·정비사업·용도지역·규제 |
| 수익성/리스크 | 임대수익률, 공실 리스크, 가격 조정 가능성 |

## 기본 산출물

- `STEP1_output.md` ~ `STEP10_output.md`: 단계별 중간 결과
- `STEP11_보고서_draft.md`: 최종 보고서 초안
- `images/`: 인과도, KPI/판단 트리 등 시각 자료
- 최종 DOCX: `05_output/[target_id].docx`
