# STEP 3: 가격·거래 추이

## 목적
대상 시장이 현재 사이클의 어느 위치에 있는지 객관적으로 파악한다.
가격과 거래량의 흐름을 수치로 제시하고, 외부 요인(금리·정책)과의 인과관계를 연결한다.

## 입력
- STEP1_output.md (기준 시점, 대상 범위)
- STEP2_output.md (금리·정책 환경)
- 국토교통부 실거래가 공개시스템
- KB부동산 시세
- 한국부동산원 R-ONE (거래량, 전세가율)

## 작업 체크리스트
- [ ] 실거래가 시계열 (최근 3~5년, 분기 또는 반기 단위)
- [ ] 거래량 변화 (전년 동기 대비)
- [ ] 전세가율 또는 임대수익률 추이
- [ ] 가격 변동과 금리·정책 변화의 상관관계 서술
- [ ] 고점 대비 현재 수준 (상승폭·하락폭)
- [ ] 각 수치에 출처와 기준 시점 표기

## 산출물
`06_middle_output/[target_id]_KR/STEP3_output.md`

### 포맷
```markdown
# STEP3 가격·거래 추이

## 1. 실거래가 흐름 (표 또는 수치 나열)
## 2. 거래량 변화
## 3. 전세가율
## 4. 외부 요인과의 관계
## 5. 현재 시장 위치 진단
```
