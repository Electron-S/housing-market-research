# STEP6 입지 분석

## 1. 교통 접근성

### 청량리역 (도보 5~10분 이내)

청량리역은 수도권 내 최다 노선이 집결하는 광역 교통 허브다.

| 노선 | 주요 행선지 | 현황 |
|---|---|---|
| 1호선 | 광화문·서울역·용산 (30분 이내) | 운행 중 |
| 경의중앙선 | 왕십리·공덕·수색 | 운행 중 |
| 경춘선 | 춘천 방면 | 운행 중 |
| 수인분당선 | 왕십리·선릉·수원 | 운행 중 |
| GTX-B | 여의도·용산·서울역 | 착공 진행 중 |
| GTX-C | 삼성역(1정거장)·양재·수원 | 착공, **지연 가능성 있음** |
| 동북선 경전철 | 왕십리 방면 | 2026년 완공 예정 |

> 출처: 각 철도 노선 운영기관, 2026.04 기준

**GTX 유의사항:** GTX-C의 청량리역 구간은 공사비 증액 협상 난항으로 2028년 개통 목표에 차질 우려. 호재로 반영하되 확정 전 단정 금지.

### 주요 업무지구 접근성 (현재 대중교통 기준)

| 업무지구 | 소요시간 | 수단 |
|---|---|---|
| CBD (광화문·을지로) | 약 25~30분 | 1호선 직통 |
| GBD (강남·삼성) | 약 40~50분 (현재) / 10분 이내 (GTX-C 개통 시) | 지하철·GTX |
| YBD (여의도) | 약 40분 (현재) / 15분 (GTX-B 개통 시) | 지하철·GTX |
| 용산 | 약 25분 | 경의중앙선 |

## 2. 생활 인프라

### 의료
- 고려대학교 안암병원 (약 1km)
- 경희의료원 (약 1.5km)
- 서울성심병원

→ 의료 접근성 최상위권. 의료 종사자 및 중장년 실수요층의 주거 선호 요인.

### 자연환경
- 홍릉수목원 인접 (영구 조망권 가능) → **숲세권** 프리미엄
- 정릉천 산책로

### 상업·편의
- 청량리역 상권 (대형마트·롯데백화점·전통시장 밀집)
- 경동시장 인근

### 교육
- 초등: 청량 초·전농 초 (배정 구역 확인 필요)
- 중·고: 동대문구 학군 (강남 대비 열위이나 개선 기대)
- 대학: 고려대, 경희대, 서울시립대 도보·자전거 거리

## 3. 개발 호재 (확정 또는 진행 중 사항)

| 호재 | 내용 | 상태 |
|---|---|---|
| 동북선 경전철 | 왕십리 연결, 청량리역 환승 | 2026년 완공 예정 |
| GTX-B | 여의도·용산·서울역 직결 | 착공 중 |
| GTX-C | 삼성역 1정거장 | 착공, 지연 우려 |
| 홍릉 바이오의료 클러스터 | 첨단 의료·연구 업무지구 개발 | 진행 중 |
| 동대문구 정비사업 | 30,647세대 계획, 지역 이미지 전환 | 진행 중 |

> GTX-C는 추정 수혜로 표기. 확정 개통 전까지 분양가 산정 시 보수적으로 반영할 것.

## 4. 수요층 관점 입지 총평

**강점**
- CBD 직주근접 + 다수 노선 → 30~40대 맞벌이 직장인 수요 흡수력 높음
- 대형 의료기관 밀집 → 의료 종사자 고정 임차·매수 수요
- 홍릉수목원 조망권 + 청량리역 상권 → 생활편의성·환경 동시 충족
- 신축 희소 → 일반분양 물량 527세대는 충분히 소화 가능한 규모

**약점**
- 학군 경쟁력: 강남·목동·노원 대비 열위. 초등 배정 구역 및 중학군 확인 필요
- GTX 불확실성: 호재 선반영 시 실현 지연 리스크 존재
- 청량리역 북측 노후 상권 이미지 개선 시간 소요
